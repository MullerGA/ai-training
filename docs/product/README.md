# Vue d'ensemble Produit

## Positionnement

AI TRAINING est une application de formation IA centrée sur le parcours apprenant. Elle propose une navigation claire par étapes: découverte, apprentissage, pratique, suivi de progression.

## Portée actuelle

- Expérience learner multi-écrans.
- Données métier statiques (mockées localement).
- UI moderne basée sur composants réutilisables.
- Application prête à être branchée à une API plus tard.

## Personas ciblés

- Collaborateurs qui découvrent l'IA générative.
- Managers souhaitant suivre la montée en compétences.
- Équipes souhaitant pratiquer le prompting.

## Proposition de valeur

- Réduire le temps d'entrée sur les sujets IA.
- Guider l'apprenant avec des modules progressifs.
- Rendre la progression visible et motivante.
